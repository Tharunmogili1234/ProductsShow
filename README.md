This is a complete runnable project (frontend + backend) built with:
- React (frontend)
- Node.js + Express (backend)
- SQLite for data storage

## Quick start

### Backend
1. Open a terminal and run:
   ```
   cd backend
   npm install
   npm run seed
   npm start
   ```
   Server will start on http://localhost:3001

### Frontend
1. In a new terminal:
   ```
   cd frontend
   npm install
   npm start
   ```
   Frontend will start on http://localhost:3000 and proxy API calls to the backend.
